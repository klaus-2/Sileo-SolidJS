## [0.1.0] - 2026-04-05

### Added

- Initial release of sileo-solid
- Physics-based toast notifications with spring animations
- Swipe to dismiss support
- Header layer transitions
- Auto expand/collapse on hover or timed reveal
- Promise support (loading → success/error flow)
- Theme aware (light/dark/system)
- Pure SolidJS + motion DOM — zero React dependencies
